# Firebase Functions

This project uses Firebase functions for speech synthesizing over the Google TTS API.
All function code is stored in `/functions`.

## Deploy Code to Production

To deploy the code, run `firebase deploy --only functions,firestore:rules,firestore:indexes` in the terminal.
The output should look like this:
